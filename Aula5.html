O que eu gosto

Eu gosto de programar em backend e também explorar várias linguagens de programação. Entre as minhas preferidas estão:

  • Python – pela sua simplicidade, clareza e poder nas automações e servidores
  • JavaScript – especialmente com Node.js para aplicações web e APIs
  • JSON – essencial para trabalhar com dados e integração entre sistemas
  • HTML e CSS – para estruturar e estilizar páginas da web
  • SQL – para manipulação e consulta de bancos de dados
